# Expense Tracker

A simple web-based Expense Tracker application that allows users to record income and expenses, categorize expenses, and view total income, total expense, and remaining balance.

## Features
- Add transactions (Income / Expense)
- Expense category selection (Food, Travel, Shopping, Bills, Rent, Others)
- Displays:
  - Total Income
  - Total Expense
  - Total Balance
- Delete transactions
- Data stored using browser Local Storage (remains after refresh)

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Project Structure
home.html # Home page
home.css # Home page styling
expense.html # Expense tracker page
expense.css # Expense tracker styling
expense.js # Main functionality (add/delete/save transactions)
imgs/ # Images used in UI




## How to Run
1. Download or clone this repository.
2. Open `home.html` in a web browser.
3. Enter your name and click **Login** to go to the expense tracker page.

## Notes
- All records are saved in Local Storage.
- Clearing the browser storage will remove saved transactions.
